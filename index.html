    Bellringing soundscapes for Boodler

    Project maintained by alexhunsley Hosted on GitHub Pages — Theme by mattgraham

    The sound of change ringing -- for pleasure and education

    TinTinn is a soundscape package created for Boodler, a versatile soundscaping tool.

    With TinTinn you can

    • Listen to methods rung on different recorded sets of bells

    • Mix the sound of ringing with other ambient sounds: wind, rain, birds, thunder, etc.

    • Listen to ringing with either perfect striking or humanised, non-perfect striking - you choose

    • Listen to randomly selected methods - there's even a "practice night" sound agent which recreates the ringing you might typically hear

    • Train your ear by listening to ringing containing at least one imperfectly struck bell. Can you tell which bell is out? Is it late or early?

    How does TinTinn differ from Mobel and similar tools?

    • TinTinn doesn't let you take part in the ringing - it's for listening only

    • Boodler and TinTinn are open source and cross-platform. You can build on top of TinTinn and mix it up.

    • TinTinn is run from the command line

    Installing TinTinn

    First install Boodler.

    Next install the TinTinn soundscape and samples packages:

    $ boodle-mgr install snds
$ hoodle-mgr install sndscape

    Now you're ready to use TinTinn.

    Try playing a single sequence of bells:

    $ boodler com.alexhunsley.bellringing/BellSequence 13572468

    You should hear Queens.
